## Лабораторная работа №1

**Цель:**  
Познакомиться с базовыми возможностями Python: простыми типами данных, условиями, функциями, списками, словарями и основами ООП. Научиться писать и проверять функции с помощью `assert`.

**Что было сделано:**

- Заданы переменные всех основных типов: `int`, `float`, `str`, `bool`, `list`, `tuple`, `set`, `dict`.
- Реализованы функции:
  - арифметика и условия (`difference_of_two_numbers`, `condition_function`, `calculator`);
  - работа с коллекциями (`number_of_unique_elements`, `counter`, `multiply_nums`);
  - две математические функции (`custom_function`, `custom_function_1`) с использованием модуля `math`;
  - задачи на работу со временем (`time_converter`) и на поиск минимума из трёх чисел (`min_of_three_values`).
- Выполнены задачки на строки и списки:
  - удаление символа и каждого третьего символа из строки (`remove_symbol`, `remove_each_third_sym`);
  - поиск максимального элемента и его индекса, добавление элемента в список, подсчёт количества уникальных значений (`find_max`, `append_to_list`, `number_unique_elements`).
- Написаны классы:
  - `Vehicle` и наследник `Truck` с атрибутами максимальной скорости и пробега;
  - `MyList` c методами для подсчёта суммы, разворота и слайсинга списка.
